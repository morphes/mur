  Wi-Fi® Network Controller Module for M2M Market
 
 
Murata part number: 88-00151-00

Enables Simple Serial-to-Wi-Fi® Solution
► The SN8200 Wi-Fi® Network Controller Module is a small, low-power, self-contained, certified Wi-Fi® network controller module that provides simple serial-to-Wi-Fi® connectivity to the Internet. It is an ideal solution for manufacturers who are looking to add wireless connectivity to home appliances or other devices for the healthcare equipment, industrial control, and monitoring markets.

Enabled with Broadcom Corporation's Wireless Internet Connectivity for Embedded Devices (WICED™), additional integrated features of the SN8200 (P/N 88-00151-00) are built-in TCP/IP network stack, security supplicant, and other network application elements such as web server. The module provides Murata's Simple Network Interface Card (SNIC) serial-to-Wi-Fi® interface protocol that treats the Wi-Fi® module as a simple serial port, reducing the development and integration effort for design engineers and shortening the time-to-market.

The SN8200 is the first module from MWS's Network Controller Module product line. The family is composed of footprint-compatible modules with flexible software personalities and hardware configurations to match customer needs. Variants include external antenna, microcontroller speed, flash memory size, peripheral interfaces, and future advanced software feature options like Smart Energy Profile 2.0 and IPv6.

Product Features
•2.4GHz IEEE 802.11b/g/n radio technology
•Wide link budget (113dB) for long-distance coverage
•On-board antenna and TCXO
•Host interface: UART, SPI
•Extended operating temperature and product life
•FCC/IC certified; CE compliant
•Built-in TCP/IP network stack and WPA/WPA2 PSK security supplicant
•Support AP/STA dual mode and web server in AP mode
