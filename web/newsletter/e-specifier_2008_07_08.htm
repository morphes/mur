NEW PRODUCT - Low Profile VRM11.0
Ultra-low profile 80A voltage regulator module has industry-leading installed height of just 0.807in (20.5mm)

  • Low profile SIP ideal for space restricted applications
  • Meets Intel® VRM 11.0 requirements
  • 80A maximum output with DAC programmable output voltage
  • 88% full load efficiency

Murata Power Solutions has added a new model to its VR110 Series of single inline packaged (SIP) voltage regulator modules (VRMs) to address applications where component height is restricted. The VR110B080CA-1C is housed in a package that achieves an industry-leading installed height of just 0.807in. (module height is 0.782in. [19.86mm]).

The VR110B080CA-1C has a single 7-Bit DAC programmable output of 0.8375VDC to 1.6VDC; maximum output current is 80A. Input voltage is 12VDC and full load efficiency is 88%. The device is Intel® VRM 11.0 compliant.

Key features of the new module include power good output, differential remote sense, remote enable, dynamic VID capability, and a tri-state output when disabled. A comprehensive set of supervisory and self-protection features such as output over-current, short circuit protection, an output current level signal and an over-temperature indicator are also incorporated in the design.

The new module is RoHS compliant and has an overall length and width of 3.6in. x 0.75in (91.44 x 19.1 mm). Operating temperature range is 0ºC to +65ºC.

Why would I specify the VR110B080CA-1C?

  • Ultra-low profile - fits where other VRMs can't.
  • High efficiency and competitive pricing.
  • Part of the VRM11.0 series of proven power solutions.
